# VGT Sign Converter

Convert VGT-oriented text input into a timed gesture plan and play it with a web 2D avatar.

## Quick Start

- API: `cd api && npm install && npm run dev`
- Web: `cd web && npm install && npm run dev`

## Scope (v1)

- Text normalization and token/phrase mapping
- Fallback handling for unknown words
- Timed gesture plan JSON output
- 2D avatar playback with timeline controls
